ECOSYSTEM CLASS (ANNUAL CYCLE)

"A draft for the ecosystem class"

class Ecosystem

	def __init__(self, ):

	def feeding
		1. herbivores eat
		2. carnivores eat
		3. growth of fodder at beginning of year

	def procreation
		1. animals give birth
		2. calculating probability of birth with equation
			min(1, γ * Φ * (N-1))

			Number of animals N at the start of the breeding season (newborn does not count)

	def migration
		- migrate to neighbouring cells, at most once per year. Animals have special properties

	def aging
		- age += 1 for all animals in all cells

	def loss_of_weight
		- all animals lose weight according to equations (section 2.2)
